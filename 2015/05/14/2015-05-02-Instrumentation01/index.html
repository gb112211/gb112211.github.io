Instrumentation 特点

  • 该框架基于 JUnit，因此既可以直接使用 Junit 进行测试，也可以使用 Instrumentation 来测试 Android 组件
  • 其为 Android 应用的每种组件提供了测试基类
  • 可以在 Eclipse 中方便地创建 Android Test Project，并将 Test Case 直接以 Android JUnit 的方式运行
